# PassForge

Free password generator. Generate strong, secure passwords instantly. No login or installation needed. Works in any browser.

## Features
- Generate strong passwords with custom settings
- Choose length, uppercase, lowercase, numbers and symbols
- Password strength meter
- Bulk generate up to 10 passwords
- Passphrase generator
- PIN code generator
- Password history
